## Naudojimas
```console
./compile uzduotis-lyga Versija
./test uzduotis-lyga Versija [testo#]
```
Galima kompiliuoti ir testuoti viena komanda:
```console
./comptest uzduotis-lyga Versija [testo#]
```

```console
./compile rogutes-jau Oficialus
./test rogutes-jau Oficialus
```

